# 📊 Real Estate MLS Listings & Sales Analysis

## Overview
This project analyzes real estate listing and sales data to identify trends in pricing, market activity, and property characteristics. The goal is to uncover insights that support data-driven decisions in the housing market.

## Data Source
- IDX Exchange (CRMLS real estate data)

## Tools Used
- Python (pandas, numpy, glob, matplolib, seaborn)
- Tableau

## Key Analysis
- Price trends over time
- Comparison of listing vs sold prices
- Price per square foot analysis
- Days on market trends
- Property characteristics (beds, baths, sqft)

## Key Insights
- Identified differences between listing and closing prices
- Observed seasonal patterns in housing activity
- Highlighted regions with higher price per square foot
